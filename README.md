# Porter

![Porter](web/src/logos/porter-refine-main.png)

Universal agent orchestrator for GitHub Issues. Porter brings the @mention workflow to any coding agent by running tasks in Modal containers and returning PRs automatically.

## What it does

- Dispatch tasks from GitHub issues or the dashboard.
- Run agents in cloud containers (Modal) with repo context.
- Track progress, logs, and PR output in a single UI.

## Local development

```bash
cd web
bun install
bun run dev
```
